# Miniature Golf

A lightweight, web-based miniature golf game.

## Overview

This project is a simple implementation of miniature golf using HTML5 Canvas and JavaScript. It features basic physics, course layouts, and a scoring system.

## Technologies Used

- **HTML5**: For the game canvas and structure.
- **CSS**: For basic styling and layout.
- **JavaScript**: For game logic, physics, and rendering.

## How to Play

1.  Clone this repository.
2.  Open `index.html` in your favorite web browser.
3.  Use your mouse to aim and click to hit the ball. The further away from the ball you click, the harder the hit will be (if the game logic supports that). *Note: Game mechanics may vary based on `game.js` implementation.*
